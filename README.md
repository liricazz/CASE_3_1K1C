## 📋 Описание проекта

**Contact Manager** — это полнофункциональное консольное приложение для управления базой контактов физических лиц. Программа обеспечивает полный цикл работы с данными: создание, чтение, обновление, удаление (CRUD), а также расширенные возможности поиска и экспорта.
Задача решена компактно студентом Передовой Инженерной Школы РХТУ им. Д. И. Менделеева, Кондратовой Мартой.

### Решаемые задачи

- Организация структурированного хранения контактов
- Быстрый поиск по различным критериям
- Персистентность данных через файловое хранилище
- Удобный консольный интерфейс с русскоязычным меню

## Затрагиваемые темы

| Тема | Реализация в проекте |
|------|---------------------|
| **Работа с файлами** | Текстовое хранение с экранированием в `contacts.dat` |
| **Поиск по имени/номеру** | Регистронезависимый поиск через `containsCI()` |
| **Структуры данных** | `struct Contact`, `std::vector<Contact>` |
| **Ввод/вывод с консоли** | `std::cin`/`std::cout` с валидацией |
| **Меню пользователя** | Интерактивное меню с 9 опциями |
| **Командные строки** | Поддержка `--file` аргумента |

## Функциональные возможности

### Основные операции
- ✅ **Добавление контакта** — 9 полей (ФИО, телефон, email, адрес, дата рождения, заметки)
- ✅ **Просмотр всех контактов** — табличный вывод с сортировкой по фамилии
- ✅ **Просмотр по ID** — детальная карточка контакта
- ✅ **Редактирование** — изменение любого поля с сохранением существующих значений
- ✅ **Удаление** — с подтверждением операции

### Расширенные функции
-  **Многорежимный поиск**:
  - По имени/фамилии/отчеству
  - По номеру телефона
  - По email
  - По адресу
  - Универсальный поиск по всем полям
  - Экспорт в текстовый файл — читаемый формат
  - Статистика — количество контактов, отсутствующие телефон/email
  - Автосохранение — каждое изменение мгновенно сохраняется в файл

## Структура контакта

```cpp
struct Contact {
    int id;                 // Уникальный идентификатор
    std::string firstName;  // Имя
    std::string lastName;   // Фамилия
    std::string middleName; // Отчество
    std::string phone;      // Телефон
    std::string email;      // Email
    std::string address;    // Адрес
    std::string birthday;   // Дата рождения (ДД.ММ.ГГГГ)
    std::string notes;      // Заметки
};
```


## Установка и запуск

### Linux / macOS

```bash
# Компиляция
g++ -std=c++17 -o contact_manager contact_manager.cpp

# Запуск (файл по умолчанию: contacts.dat)
./contact_manager

# Запуск с указанием пользовательского файла
./contact_manager --file mydata.dat
```

### Windows

```bash
# Компиляция
g++ -std=c++17 -o contact_manager.exe contact_manager.cpp

# Запуск
contact_manager.exe

# Запуск с указанием пользовательского файла
contact_manager.exe --file mydata.dat
```
